Historias de Usuario

Historia de Usuario 1: Conexión y Control Remoto

Título: Conexión y Control Remoto del Brazo Robótico

Como usuario,

quiero poder conectar mi dispositivo al brazo robótico mediante Bluetooth,

para controlar sus movimientos remotamente y realizar la recolección de residuos.

Criterios de aceptación:

  • El usuario puede iniciar la conexión Bluetooth desde la interfaz de control remoto (Página web).
  • La interfaz indica si la conexión fue exitosa.
  • El usuario puede enviar comandos de movimiento al brazo robótico.
  • El brazo robótico responde a los comandos en tiempo real.

Historia de Usuario 2: Visualización en Tiempo Real

Título: Visualización en Tiempo Real del Área de Trabajo

Como usuario,

quiero visualizar en tiempo real el área de trabajo del brazo robótico mediante una cámara, visualizarlo en la web,

para poder identificar y recoger residuos con precisión.

Criterios de aceptación:

  • La transmisión de video en la web se inicia automáticamente al conectar con el brazo robótico.
  • El video se muestra en la interfaz de control remoto.
  • La calidad del video es suficiente para identificar objetos claramente.
  • La transmisión de video tiene un retardo mínimo.

Historia de Usuario 3: Recolección de Residuos

Título: Recolección de Residuos

Como usuario,

quiero poder mover el brazo robótico para recoger residuos,

para limpiar el área de trabajo de manera eficiente.

Criterios de aceptación:

  • El usuario puede mover el brazo robótico hacia el residuo mediante controles remotos.
  • El brazo robótico puede agarrar y
  • El brazo robótico puede agarrar y levantar residuos.
  • El sistema notifica al usuario cuando el residuo ha sido recogido con éxito.

Historia de Usuario 4: Medición del Peso del Residuo

Título: Medición del Peso del Residuo

Como usuario,

quiero medir el peso del residuo recogido,

para clasificarlo, guardar la información y gestionar su disposición adecuada.

Criterios de aceptación:

  • El brazo robótico mide el peso del residuo después de recogerlo.
  • El peso del residuo se muestra en la interfaz de control remoto.
  • El sistema es capaz de calibrar el sensor de peso cuando sea necesario.

Historia de Usuario 5: Registro de Residuos

Título: Registro de Residuos Recolectados

Como administrador del sistema,

quiero registrar la información de los residuos recolectados,

para generar informes y analizar datos de recolección.

Criterios de aceptación:

  • El sistema registra cada residuo recogido, incluyendo su peso y tipo.
  • El usuario puede acceder a un historial de residuos recolectados.
  • El sistema genera informes detallados sobre la recolección de residuos.
